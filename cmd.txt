---------------------------------------------------------
命令            简介
---------------------------------------------------------
ls             list: 查看目录结构
ls -l          long list: 详细查看
ls -a          all list: 查看所有文件(隐藏文件)
ls -i          list inode: 查看文件i节点(文件根本标记)
ls -R          list recursively: 递归地查看子目录
ls -d          list directory: 查看目录本身
---------------------------------------------------------
cd             change directory: 切换目录
---------------------------------------------------------
mkdir DIR      make directory: 创建目录
mkdir -p DIR   make parent directory: 创建子目录同时创建父目录
---------------------------------------------------------
rmdir DIR      remove directory: 移除空目录
---------------------------------------------------------
cp  s  d       copy: 拷贝文件/目录(文件属性更新)
cp -p s d      copy preserve: 保持文件同步拷贝(时间戳不变)
---------------------------------------------------------
mv s   d       move: 剪切文件/目录或者更名
---------------------------------------------------------
rm             remove file: 删除文件
rm -r          remove dir: 删除目录
rm -f          remove forcely: 强制删除
---------------------------------------------------------
touch FILE     touch: 创建文件
---------------------------------------------------------
cat  FILE      print file: 打印文件内容
cat -n FILE    print file with number: 显示行号
---------------------------------------------------------
tac  FILE      反向打印
---------------------------------------------------------
more           分页显示,空格翻页,enter换行(向下)
---------------------------------------------------------
less           分页显示,空格翻页,enter换行,PgUp回溯,上箭头换行,'/关键字'查找, n切换关键字段
---------------------------------------------------------
head           显示前部分
head -n 10     显示前10行
---------------------------------------------------------
tail           显示末尾部分
tail -n 10     显示最后10行
tail -f a.txt  动态显示末尾部分(log日志输出)
---------------------------------------------------------
ln -s 源文件 目标文件  软链接,类似WIN快捷方式,lrwxrwxrwx
ln 源文件 目标文件     硬链接,类似cp -p + 实时更新,两者inode完全一致,删除一个,还可以用另一个
                              不可以跨区,不可以创建目录的硬链接
---------------------------------------------------------
